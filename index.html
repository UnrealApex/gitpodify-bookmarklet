Gitpodify

A handy bookmarklet to quickly open a GitHub, GitLab, or Bitbucket repository in a containerized environment.

What is Gitpod?

Gitpod is an open source platform for automated and ready-to-code development environments that blends into your existing workflow. It enables developers to describe their dev environment as code and start instant and fresh development environments for each new task directly from your browser. For more information check out the Gitpod docs.

Why did I create this?

I created this bookmarket because I thought the Gitpod browser extension wasn't practical because the same result could be achieved using a bookmarklet with no installation necessary.

Getting Started

  1. To get started using Gitpodify, drag this image to your bookmarks bar.

  2. After dragging the bookmarklet to your bookmarks bar, go to any GitHub, Gitlab, or Bitbucket repository and click the bookmarklet.

    notes: if you execute this bookmarlet with any other website you will get a cannot parse error from Gitpod. if this is the first time you are doing this, you will be prompted to log in.

  3. It should take about a minute for the environment to load. Once its done, congratulations! you've opened a repo with Gitpod using the Gitpodify.
  4. Now you can contribute to open source right from the browser
