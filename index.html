Open with Gitpod

A handy bookmarklet you can use to open a GitHub, GitLab, or Bitbucket repository in a containerized environment

What is Gitpod?

Gitpod is an open source platform for automated and ready-to-code development environments that blends into your existing workflow. It enables developers to describe their dev environment as code and start instant and fresh development environments for each new task directly from your browser. For more information check out the Gitpod docs.

Getting Started

  1. To get started using Gitpod, first drag this image to your bookmarks bar Open with Gitpod
  2. After dragging the bookmarklet to your bookmarks bar, go to any GitHub, Gitlab, or Bitbucket repository and click the bookmarklet. If this is the first time you are doing this, you will be prompted to login.
