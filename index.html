Gitpodify

A handy bookmarklet to quickly open a GitHub, GitLab, or Bitbucket repository in Gitpod.

Getting Started

  1. To use the bookmarklet, drag this image to your bookmarks bar:

  2. After dragging the bookmarklet to your bookmarks bar, go to any GitHub, Gitlab, or Bitbucket repository and click the bookmarklet.

Note that using this bookmarklet creates a new workspace each time you use it for a respository
